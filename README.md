## ◈ About

<p align="left">
I am a third-year Information Sciences (SI2) student at the Faculty of Sciences and Techniques of Sbeitla, University of Kairouan, Tunisia — engineering robust, scalable software across the full stack. My academic and professional trajectory converges on a single ambition: building products that matter, with engineering discipline that lasts.

My PFE internship at <strong>Astrée Assurances</strong> (BIAT Group) delivered a production-grade hybrid insurance platform integrating <strong>Flutter</strong>, <strong>Angular</strong>, <strong>Firebase</strong>, and <strong>Google Gemini AI</strong> — spanning mobile UX, real-time backend orchestration, and AI-powered claim automation. I operate at the intersection of product thinking and systems engineering.

I approach every problem through the <strong>2TUP methodology</strong> — iterative, architecture-first, test-validated. My toolchain spans agile project management (Jira, Scrum), infrastructure-as-code, CI/CD pipelines, and AI integration. Outside engineering, I study Japanese language and culture, and hold a deep interest in cross-cultural product design for global markets.
</p>

## ◈ Featured Projects

<details>
<summary><strong>⬡ Hybrid Insurance Platform — Astrée Assurances (BIAT Group)</strong></summary>
<br>

A full-cycle, production-grade insurance management platform built as a final-year engineering project (PFE) for Astrée Assurances, a subsidiary of BIAT Group — Tunisia's largest banking institution.

The platform delivers a unified cross-platform experience across mobile (Flutter) and web (Angular), backed by Firebase real-time infrastructure and augmented by Google Gemini AI for intelligent claims processing, document analysis, and client advisory automation.

| Attribute | Detail |
|---|---|
| **Stack** | Flutter · Angular · Firebase · Google Gemini AI · Firestore |
| **Scale** | Enterprise — multi-actor, multi-platform, multi-region |
| **Performance** | Real-time sync, sub-200ms response on critical workflows |
| **Security** | Firebase Auth, role-based access control, encrypted storage |
| **Impact** | Automated claims intake, AI-powered document parsing, reduced manual processing |
| **Repository** | [github.com/Benjidagoat-maker](https://github.com/Benjidagoat-maker) |

The platform was designed and built following the **2TUP (Two-Track Unified Process)** methodology, delivering both the functional and technical branches in parallel — UML diagrams, use-case modeling, sequence diagrams, and component architecture all validated prior to implementation.

</details>

<details>
<summary><strong>⬡ Federated Project Management Platform — FST-SBZ</strong></summary>
<br>

An enterprise-grade academic project management platform built for the Faculty of Sciences and Techniques of Sbeitla, supporting four distinct actor roles: Student, Tutor, Coordinator, and Jury.

The system was engineered with a full Scrum backlog (6 Epics, 39 User Stories across 4 Sprints), Jira project management, GitHub Actions CI/CD, and BDD acceptance testing using Playwright/TypeScript — migrated mid-cycle from Firebase to Supabase for production stability and relational data integrity.

| Attribute | Detail |
|---|---|
| **Stack** | Supabase · PostgreSQL · GitHub Actions · Playwright · TypeScript |
| **Scale** | 4 actor roles · 39 user stories · 4 sprints · Full BDD test coverage |
| **Performance** | Relational query optimization, real-time subscription layer |
| **Security** | Supabase RLS policies, JWT-based auth, role-scoped access |
| **Impact** | Full Scrum lifecycle automation — tickets auto-synced Jira ↔ GitHub |
| **Repository** | [github.com/Benjidagoat-maker](https://github.com/Benjidagoat-maker) |

GitHub Actions workflows were implemented to synchronize Jira ticket state with GitHub pull request and commit events in real time, using direct Jira REST API calls via curl — eliminating manual sprint tracking overhead.

</details>

<details>
<summary><strong>⬡ Interactive Japanese Language Learning Platform</strong></summary>
<br>

A full curriculum-mapped Japanese language learning web application, architected around the CEFR (A1–B2) and JLPT (N5–N3) frameworks — built as a solo full-stack project with a dark aesthetic and structured pedagogical progression.

| Attribute | Detail |
|---|---|
| **Stack** | React · Tailwind CSS · Custom SVG animations |
| **Scale** | Full A1–B2 curriculum · Hiragana → Kanji → Grammar → JLPT |
| **Performance** | Client-side rendering, zero-latency lesson transitions |
| **Security** | Static SPA, no PII collected |
| **Impact** | Structured self-directed language learning tool with quiz scoring |
| **Repository** | [github.com/Benjidagoat-maker](https://github.com/Benjidagoat-maker) |

Designed with a dark Japanese aesthetic — gold and sakura accents — and a full interactive quiz engine with animated scoring. Curriculum spans Hiragana mastery, vocabulary, grammar structures, and JLPT-aligned reading comprehension.

</details>
